# fishingGame: A ChonGame Project
A simple project made with chonGame, a JavaFX framework.

> [!NOTE]
> Please check the [original repository](github.com/profpantoja/chonGame) for clear instructions on how to run it.

# How To Play?
It's a very simple game, simply press the **SPACE KEY** to throw your rod, after that pay close attention to the keys that will appear on screen, if you manage to press them in time you get a fish (and earn 1 point!)
